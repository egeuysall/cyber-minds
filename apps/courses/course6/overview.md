# Course 6 - Linux Fundamentals

## Course Overview

This course introduces Linux, the go-to operating system for cybersecurity professionals. You'll learn what Linux is, how to set it up, and the basic commands needed to navigate and use this powerful command-line interface.

## Lessons

1. **Lesson 1: Introduction** - What is Linux and why use it for cybersecurity
2. **Lesson 2: Setting Up Linux** - Installing VirtualBox and a Linux distribution
3. **Lesson 3: Basic Commands and Getting Help** - Essential Linux commands and how to find help
4. **Lesson 4: Sudo & Privilege Escalation** - Understanding permissions and the file system

## Learning Objectives

By the end of this course, you will:

- Understand what Linux is and its importance in cybersecurity
- Set up a virtual Linux environment using VirtualBox
- Navigate the Linux file system using basic commands
- Learn how to get help with Linux commands
- Understand file permissions and privilege escalation
