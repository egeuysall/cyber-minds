# Course 9 - Penetration Testing

## Course Overview

This course introduces penetration testing (ethical hacking) - the practice of legally testing systems to find vulnerabilities before malicious hackers do. You'll learn about different types of hackers, reconnaissance techniques, and pentesting tools and concepts.

## Lessons

1. **Lesson 1: Introduction** - What is penetration testing and ethical hacking
2. **Lesson 2: Hackers & Boxes** - Types of hackers and testing environments
3. **Lesson 3: Reconnaissance** - Information gathering and target analysis
4. **Lesson 4: Resources for Pentesting** - Tools, platforms, and learning resources
5. **Lesson 5: Pentesting Concepts** - Key methodologies and techniques

## Learning Objectives

By the end of this course, you will:

- Understand what penetration testing is and its importance
- Learn about different types of hackers (white hat, black hat, gray hat)
- Master reconnaissance techniques for information gathering
- Discover tools and resources for penetration testing
- Learn key pentesting concepts and methodologies
