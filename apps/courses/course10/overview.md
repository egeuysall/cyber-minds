# Course 10 - Networking Fundamentals

## Course Overview

This course covers the essential networking concepts that are crucial for understanding cybersecurity. You'll learn about network types, protocols, addressing, network models, topologies, and security methods.

## Lessons

1. **Lesson 1: Introduction to Networking** - What is a network and why it matters for cybersecurity
2. **Lesson 2: Networking Basics** - Network types, devices, and terminology
3. **Lesson 3: Protocols & Ports** - How data is transmitted and services communicate
4. **Lesson 4: Addresses & Packets** - IP addressing and data packaging
5. **Lesson 5: The OSI & TCP/IP Models** - Understanding network communication layers
6. **Lesson 6: Network Topologies** - Different network structures and configurations
7. **Lesson 7: Network Security Methods** - Protecting networks with various security technologies

## Learning Objectives

By the end of this course, you will:

- Understand fundamental networking concepts and terminology
- Learn about network protocols and how they function
- Master IP addressing and packet transmission
- Understand the OSI and TCP/IP models
- Explore different network topologies
- Discover network security methods and technologies
